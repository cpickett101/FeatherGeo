# Changelog

All notable changes to FeatherGeo will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Browser-based GIS tool with no server requirements
- Import support for Shapefiles (.shp with sidecars or .zip) and GeoJSON
- Interactive map with feature inspection via click
- Spatial operations: Buffer, Simplify, Centroid, Convex Hull, Bounding Box
- Feature selection (single and multiple)
- Export to GeoJSON or Shapefile (.zip)
- Metric and imperial unit support
- Auto-save last session in browser storage
- Built with React, OpenLayers, and GDAL (WebAssembly via gdal3.js)
